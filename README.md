# FlipTrack 🎯

Welcome to **FlipTrack** — a modern, lightning-fast web application built for an exceptional user experience. This project leverages the latest web technologies to deliver a snappy, responsive, and beautiful interface straight out of the box.

## 🚀 Key Features

- 🎨 **Beautiful UI Out of the Box**: A completely custom, polished, and accessible interface relying on custom theming tokens and components.
- ⚡ **Lightning-Fast Performance**: Built on a modern React Router (v7) setup to ensure instantaneous page loads and smooth interactions.
- 🎯 **Consistent Design System**: A unified design language governed by robust CSS variables (`app/styles/theme.css`). Customize your brand once, and see the changes reflect everywhere.
- 📱 **Fully Responsive**: Adapts flawlessly across desktop, tablet, and mobile displays.

## 🛠️ Tech Stack

- **Framework**: React Router v7
- **Styling**: Vanilla CSS with comprehensive custom properties and Radix UI primitives.
- **Language**: TypeScript
- **Tooling**: Vite for lightning-fast HMR and optimized builds.

## 🏃 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js (v18 or higher recommended) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rushikesh-bobade/FlipTrack.git
   ```
2. Navigate into the project directory:
   ```bash
   cd fliptrack
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

The application will now be running on `http://localhost:5173`.

## 📁 Project Structure

- `/app` - Contains all React application code (components, styles, routes).
  - `/app/styles` - Contains the global CSS, resets, and the foundational `theme.css`.
  - `/app/blocks` - Global layout elements like navigation bars and footers.
- `/public` - Static assets, including our custom `favicon.svg`.
